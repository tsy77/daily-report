TOP 3

Returns now supported in Customer Accounts API

shopify.dev

Shopify 让客户可以自助退货，同时支持换货、退货费用计算和退款到店铺余额——这说明商家对退货体验的需求已从"被动处理"升级到"主动设计"。退货是跨境电商最大的成本黑洞，智能退货路由是高价值 SaaS 方向。

反向视角 — 今天不要追

  • 通用 AI 图片/视频生成器 — ReCardEx、ViralCraftAI 等已有大量竞品，没有电商垂直场景差异化就不要进入
  • 跨境支付基础设施 — 市场已有 Wise、Airwallex、Stripe，需要牌照和合规能力，壁垒太高
  • 通用 SaaS 定价/获客策略 — 非电商具体场景的通用讨论，信号虽多但无法变现
  • 纯开发者工具迁移 — CLI 4.0、Polaris 迁移等是开发者内部事务，商家不会为此付费
  • 3D 模型/商品可视化 — 门槛仍然太高，中小卖家采用率极低，属于"有趣但不紧急"
